<p> Amarok è un'applicazione per Android che consente di nascondere i file e le applicazioni private con un solo clic.</p><p> In molti casi, utilizziamo sofisticati crittografi per crittografare i nostri file e le nostre applicazioni, proprio per non volerli vedere. Tuttavia, molti di questi crittografi sono lenti e richiedono molte risorse, rendendo la crittografia di file di grandi dimensioni come video e musica un compito arduo. Anche se queste soluzioni possono essere sicure, spesso sono eccessive per l'utente medio che vuole semplicemente evitare che altri accedano accidentalmente ai suoi file e alle sue applicazioni private.</p><p> Amarok è stato progettato specificamente per queste situazioni, in quanto è un "nascondiglio" leggero per file e applicazioni. Si limita a oscurare i nomi e le firme dei file, rendendoli illeggibili per l'uomo, e questo processo può essere eseguito quasi istantaneamente, indipendentemente dalle dimensioni del file. Inoltre, "disattiva" le applicazioni, nascondendole di fatto dal launcher. Tutte queste funzioni possono essere eseguite con un solo clic.</p><b>Caratteristiche</b><ul> <li><b>Nascondere file di grandi dimensioni ad alta velocità</b>: Offusca i nomi e le firme dei file per renderli illeggibili. </li> <li><b>Nascondere le applicazioni senza root</b>: Rende invisibili le applicazioni sul launcher. Attualmente supporta le modalità Root, Shizuku e DSM.</li> <li><b>UI squisita</b>: Design Material3 completo.</li></ul><p> <b><u>NOTA: AMAROK NON È UNO STRUMENTO DI CRIPTAZIONE. SI LIMITA A "OFFUSCARE" I FILE IN MODO CHE NON POSSANO ESSERE LETTI DALL'UOMO E A DISABILITARE LE APP. QUESTI PROCESSI POSSONO ESSERE FACILMENTE RIPRISTINATI. NON USARE AMAROK PER PROTEGGERE FILE IMPORTANTI! </u></b></p>
